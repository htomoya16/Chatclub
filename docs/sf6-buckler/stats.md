# SF6 Buckler Stats

本ドキュメントは、Buckler 対戦ログから算出する統計指標を定義する。

現状:

- 統計の集計/表示は **都度クエリ** で実装する

---

## 0. 勝敗の算出

- `round_results` の **値 > 0 の個数**を勝ちラウンド数として扱う
- 自分の勝ちラウンド数 > 相手 → win
- 自分の勝ちラウンド数 < 相手 → loss
- 同数 → draw

---

## 1. 基本統計（必須）

- 総試合数
- 勝・敗・引き分け
- 勝率（**引き分けを除外**: win / (win + loss)）

---

## 2. キャラ別統計

- 自キャラごとの勝率（引き分け除外）
- 自キャラごとの勝・敗・引き分け

---

## 3. セッション統計

- セッション開始〜終了までの総試合数 / 勝率 / キャラ別勝率

---

## 4. 期間別統計（任意）

- 期間内の総試合数 / 勝率 / キャラ別勝率

---

## 5. 表示方針

- Discord での表示は **簡潔な要約 + キャラ別** を基本とする

---

## 補足: 期間指定のタイムゾーン

- 日付入力は `YYYY-MM-DD`
- 解釈は **日本時間（JST）**
