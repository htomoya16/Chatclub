# SF6 Buckler Stats

本ドキュメントは、Buckler 対戦ログから算出する統計指標を定義する。

---

## 0. 勝敗の算出

- `round_results` の **値 > 0 の個数**を勝ちラウンド数として扱う
- 自分の勝ちラウンド数 > 相手 → win
- 自分の勝ちラウンド数 < 相手 → loss
- 同数 → draw

---

## 1. 通算統計

- 総試合数
- 勝率（勝利数 / 総試合数）
- 直近 N 戦勝率
- 連勝 / 連敗（最大値、現在の継続）

---

## 2. キャラ別統計

- 自キャラ × 相手キャラの勝率
- 上位（得意）組み合わせ
- 下位（苦手）組み合わせ

---

## 3. セッション統計

- セッション内の勝敗推移
- セッション内の連勝 / 連敗
- セッション開始〜終了までの総試合数 / 勝率

---

## 4. 期間別統計（任意）

- 日 / 週 / 月単位の勝率推移
- 期間内の試合数推移

---

## 5. 表示方針

- Discord での表示は **簡潔な要約 + 詳細オプション**を基本とする
- セッション中は短い間隔で更新し、結果の再掲示は控えめにする
